Write a program to implement Parallel Bubble Sort and Merge sort using OpenMP. Use existing algorithms and measure the performance of sequential and parallel algorithms.

SHORT DEFINITION:
This program uses OpenMP for parallel sorting. Bubble Sort compares adjacent elements and swaps them, while Merge Sort divides the array into smaller parts, sorts them, and merges them. Parallel processing improves sorting speed by using multiple threads simultaneously.

REAL LIFE EXAMPLE:
Suppose 100 student marks need to be arranged in ascending order.

Sequential Processing:
One teacher sorts all marks alone.

Parallel Processing:
Multiple teachers sort different parts at the same time and combine results later.

This reduces execution time.

USE CASES:
1. Database sorting
2. Data analytics
3. Search engines
4. Scientific computing
5. Machine learning
6. Big data processing

CODE:

#include <iostream>
#include <omp.h>

using namespace std;

int main()
{
    int n;

    cout << "Enter number of elements: ";
    cin >> n;

    int arr[n];

    cout << "Enter elements:\n";

    for(int i = 0; i < n; i++)
    {
        cin >> arr[i];
    }

    // Parallel Bubble Sort
    #pragma omp parallel for

    for(int i = 0; i < n-1; i++)
    {
        for(int j = 0; j < n-i-1; j++)
        {
            if(arr[j] > arr[j+1])
            {
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }

    cout << "Sorted Array:\n";

    for(int i = 0; i < n; i++)
    {
        cout << arr[i] << " ";
    }

    return 0;
}

IMPORTANT VIVA QUESTIONS:

Q1) What is Bubble Sort?
Bubble sort compares adjacent elements and swaps them if needed.

Q2) What is Merge Sort?
Merge sort divides array into smaller parts, sorts them, and merges them.

Q3) What is OpenMP?
OpenMP is an API for parallel programming.

Q4) What does #pragma omp parallel for do?
It divides loop iterations among multiple threads.

Q5) Why use parallel processing?
To improve execution speed and efficiency.

Q6) What is swapping?
Exchanging positions of two elements.

Q7) Difference between Sequential and Parallel Processing?
Sequential uses one thread, parallel uses multiple threads.

Q8) Advantage of Merge Sort?
Efficient sorting for large data.
